커스텀 도메인 연결

💡 도메인은 itcl.kr을 쓰거나, 새로 구매하거나, Vercel 무료 서브도메인을 쓸 수 있습니다. 현재 itcl.kr은 호스팅.kr에서 DNS 관리 중이고 Google Workspace 이메일에 연결되어 있습니다.
연결 절차 (덕진님 소장 요청 시 진행):
  1. Vercel Dashboard → Project Settings → Domains → Add
  2. 호스팅.kr DNS에 A 또는 CNAME 레코드 추가
  3. 현재 itcl.kr에 설정된 litt.ly/kimdukjin 포워딩 해제 필요
  4. SSL 자동 프로비저닝 대기 (~15분)
  5. api/contact.js의 SITE_URL 상수 업데이트
