[전체 작업 총평 — 강의 메시지]
================================================================================

이 세션에서 Claude Code가 한 일 (흔히 "못 한다"고 알려진 것 포함):

1. 하네스 엔지니어링 — 메타 스킬로 프로젝트 전용 에이전트 팀 + 스킬 설계
2. 멀티스크린 환경에서 PowerShell 스크린샷 기반 시각적 디버깅 (UI 탐색)
3. Vercel API 토큰 자동 추출 + REST 호출로 SSO 설정 변경
4. Supabase 신 API 키 체계의 RLS 매핑 이슈 진단 + 아키텍처 피벗 (프록시 패턴)
5. 서버리스 함수 작성 + 환경변수 기반 시크릿 관리 설계
6. CMD 복붙 제약을 감지하고 "브라우저↔브라우저" 우회 경로 설계

핵심은: Claude Code가 "명령 실행기"가 아니라 **상황 판단 → 아키텍처 결정 →
실행 → 검증**까지 주도한다는 점. 사용자 요구만으론 해결 안 되는 문제를
만났을 때 원인 분석 후 더 나은 설계로 피벗하는 것까지가 Claude의 역할.
